Summarization Chatbot

Overview

A robust chatbot built from scratch that leverages modern natural language processing techniques to generate concise summaries of any provided text. The system combines a Flask-based backend with a React frontend, and utilizes transformer models (powered by Hugging Face's Transformers and PyTorch) to dynamically adjust summary lengths based on the input content.

Key Achievements

  • Designed and implemented a full-stack solution using Flask for the API and React for the user interface.
  • Integrated advanced transformer models for state-of-the-art text summarization.
  • Implemented dynamic summary generation that adjusts to the length and complexity of the input text.
  • Enabled file uploads and maintained a chat log for enhanced interactivity.

Technologies Used:

  • Backend: Python, Flask, Flask-CORS, PyTorch, Hugging Face Transformers
  • Frontend: React, Axios, React Scripts
  • Data & NLP: CNN/DailyMail dataset, dynamic tokenization, sequence-to-sequence generation
