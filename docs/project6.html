Chess Bot Ultimate Battle

Overview

Chess Bot Ultimate Battle is an interactive chess application where players compete against a custom-built bot powered by Stockfish. The project features an impressive, modern user interface with smooth animations, a responsive chess board supporting both drag-and-drop and tap interactions, and dynamic move validation powered by chess.js. The botâ€™s difficulty can be set from a beginner-friendly 300 Elo up to 2850 Elo, allowing a tailored challenge for every skill level.

Key Achievements

  • Developed a full-stack chess bot using Node.js, Express, and vanilla JavaScript for the frontend.
  • Integrated Stockfish using the UCI protocol with strength limiting to simulate various Elo levels.
  • Implemented real-time move validation and legal move hints (displaying available moves with visual cues) using chess.js.
  • Enabled multiple user input methods including drag-and-drop and tap/click for an intuitive gameplay experience.
  • Designed a dynamic and modern user interface complete with game-over overlays, restart functionality, and customizable difficulty settings.

Technologies Used:

  • Backend: Node.js, Express, Child Process (for spawning Stockfish), UCI protocol
  • Frontend: HTML5, CSS3, JavaScript, chess.js
  • Chess Engine: Stockfish (with UCI_LimitStrength and UCI_Elo options)
  • UI/UX: Responsive design, animations, and drag-and-drop interactions
