Interactive Portfolio Website

Overview

This project is a dynamic, multi-page portfolio website designed to showcase programming projects and technical skills in an engaging, interactive format. With smooth transitions, responsive navigation, and creative visual effects, the site offers an immersive user experience.

Key Achievements

  • Developed a fully responsive, multi-page website with smooth transitions and interactive elements.
  • Integrated a theme toggle feature that adjusts colors across the entire site while keeping critical text black.
  • Implemented interactive simulations and creative design elements to enhance user engagement.
  • Ensured cross-browser compatibility and optimized performance for a seamless experience.

Technologies Used:

HTML5 & CSS3, Python (backend integration), Streamlit, JavaScript
