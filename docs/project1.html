House Price Prediction Model

Overview

This project is an end-to-end machine learning pipeline designed to predict house prices using real estate data. It incorporates thorough data cleaning, feature engineering, and the application of multiple regression models to deliver accurate predictions.

Key Achievements

  • Extracted numeric features from complex textual data using regular expressions.
  • Engineered metrics such as price per square meter to boost predictive accuracy.
  • Removed outliers using the IQR method to enhance data quality.
  • Trained and compared various regression models (Linear Regression, Decision Trees, Random Forests) using MSE and RÂ² metrics.

Technologies Used:

Python, Pandas & NumP, Scikit-learn, Matplotlib
