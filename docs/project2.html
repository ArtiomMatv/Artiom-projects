Auto ML Pipeline

Overview

This project is an end-to-end automated machine learning pipeline that streamlines the entire data science workflow. It covers everything from dataset management and feature detection to model training, evaluation, and deploymentâ€”all integrated into a single, efficient framework.

Key Achievements

  • Maintained an up-to-date requirements.txt for reproducibility.
  • Implemented an automatic feature detection module to classify features (numerical, categorical).
  • Developed an artifact management module to store and version model artifacts.
  • Designed a feature module with metadata for data preprocessing and modeling.
  • Integrated a comprehensive set of evaluation metrics with custom extensions.
  • Built a core model interface along with extensions for additional algorithms.
  • Created a pipeline evaluation component to calculate performance metrics and select the best model.
  • Developed Streamlit pages for dataset management, model selection, pipeline configuration, and deployment.
  • Added visual output using Matplotlib and Seaborn to compare trained and actual models.

Technologies Used:

Streamlit, Pydantic, Pandas, NumPy, Scikit-learn, Seaborn, Matplotlib
